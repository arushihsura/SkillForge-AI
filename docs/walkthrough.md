# YouSkill Career Intelligence: v3.1 Visual Walkthrough

This walkthrough showcases the state-of-the-art features implemented by **Team ByteWorks** for the IISc Hackathon.

## 1. Flexible Job Description Input
Users can now toggle between uploading a JD file or directly pasting raw text. This "Quick-Input" mode is essential for rapid career iteration.

## 2. YouSkill Motivation Hub
The dashboard now includes a motivational entry point. We believe upskilling is as much about mindset as it is about technical ability.

![Dashboard Motivation Card](./docs/images/motivation_card.png)

## 3. Humanized YouSkill Coach
Our Gemini-powered coach doesn't just give roadmaps—it listens. With **tone and sentiment awareness**, YouSkill detects when a user is overwhelmed and shifts to a supportive, mentoring style.

![YouSkill Coach UI](./docs/images/youskill_coach.png)

## 4. SHAP Explainability & Career ROI (v3.1)
The ML engine now provides a **13th stage of inference**: SHAP Explainability. It quantifies the precise "weight" of each skill on your readiness score, paired with Market Intelligence metrics like Competitive Percentile and Hire Probability ROI.
